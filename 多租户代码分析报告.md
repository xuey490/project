您的项目已经具备了相当完善的多租户基础架构，主要包含以下核心组件：

| 组件 | 文件路径 | 职责 |

## 二、代码质量评估

### 2.1 优点（做得好的地方）

#### ✅ 1. 架构设计合理

**TenantContext 的设计非常优秀：**

```php
// 使用静态方法管理租户状态，简单清晰
TenantContext::setTenantId(1001);   // 设置租户
TenantContext::ignore();            // 超管模式
TenantContext::restore();           // 恢复隔离
TenantContext::shouldApplyTenant(); // 判断是否应用租户隔离
```

**全局作用域自动隔离：**

```php
// LaTenantScope 自动为所有查询添加租户条件
public function apply(Builder $builder, Model $model)
{
    if (!TenantContext::shouldApplyTenant()) {
        return;
    }
    $tenantId = TenantContext::getTenantId();
    if ($tenantId && in_array('tenant_id', $model->getFields())) {
        $builder->where($model->qualifyColumn('tenant_id'), '=', $tenantId);
    }
}
```

### 2.2 存在的问题与风险

#### ⚠️ 问题 1：Workerman 环境下静态变量污染风险

**风险等级：高**

```php
// TenantContext.php 第 35-42 行
private static ?int $tenantId = null;      // ⚠️ 静态变量
private static bool $ignoreTenant = false; // ⚠️ 静态变量
```

**问题说明：**
- Workerman 是常驻进程，静态变量在请求之间不会自动重置
- 如果请求 A 设置了 tenant_id=1，请求 B 可能读取到这个值
- 这会导致严重的数据泄露问题

**复现场景：**
```php
// 请求 A：用户登录租户 1
TenantContext::setTenantId(1);
// 请求处理完成，但进程不退出

// 请求 B：新用户访问（同进程）
$tenantId = TenantContext::getTenantId(); // 可能返回 1！
```

#### ⚠️ 问题 2：用户-租户-角色关系设计不完整

**风险等级：中**

当前设计存在的问题：

```php
// SysUser.php - 用户角色关联（第 136-144 行）
public function roles(): BelongsToMany
{
    return $this->belongsToMany(
        SysRole::class,
        'sys_user_role',  // ⚠️ 没有 tenant_id 字段
        'user_id',
        'role_id'
    );
}
```

```php
// SysUserRole.php - 关联表模型（第 30-67 行）
class SysUserRole extends Model
{
    // ⚠️ 没有 tenant_id 字段
    protected $fillable = ['user_id', 'role_id', 'created_by', 'updated_by'];
}
```

**问题说明：**
- 用户可以在不同租户拥有不同角色，但当前设计无法支持
- 如果用户 A 在租户 1 是管理员，在租户 2 是普通用户，当前表结构无法区分
- 查询角色时可能返回错误租户的角色

#### ⚠️ 问题 3：SysUserRole 模型未继承 BaseLaORMModel

**风险等级：低**

```php
// SysUserRole.php 第 30 行
class SysUserRole extends Model  // ⚠️ 直接继承 Laravel Model
{
    // 缺少 BaseLaORMModel 的功能：
    // - 雪花ID生成
    // - 多租户隔离（如果需要）
    // - 时间戳自动填充优化
}
```

#### ⚠️ 问题 4：缺少租户表模型

**风险等级：中**

项目中没有找到 `Tenant` 或 `SysTenant` 模型，需要创建租户信息表模型。

#### ⚠️ 问题 5：部门、角色、菜单的租户隔离策略不明确

当前这些模型都继承 `BaseLaORMModel`，但没有明确：
- 是否所有表都需要 `tenant_id` 字段
- 公共数据（如系统菜单）如何处理
- 超级管理员的部门/角色如何设计

## 三、优化建议

### 3.1 修复 Workerman 静态变量污染（高优先级）

**方案：使用 Request 上下文替代静态变量**

```php
<?php
declare(strict_types=1);

namespace Framework\Tenant;

use Workerman\Protocols\Http\Request;

/**
 * Workerman 安全的租户上下文管理器
 */
final class TenantContext
{
    /**
     * 请求属性键名
     */
    private const TENANT_ID_KEY = '_tenant_id';
    private const IGNORE_TENANT_KEY = '_ignore_tenant';
    
    /**
     * 当前请求实例
     */
    private static ?Request $currentRequest = null;
    
    /**
     * 设置当前请求实例（由中间件调用）
     */
    public static function setRequest(Request $request): void
    {
        self::$currentRequest = $request;
    }
    
    /**
     * 设置租户ID（存储在 Request 中）
     */
    public static function setTenantId(?int $tenantId): void
    {
        if (self::$currentRequest !== null) {
            self::$currentRequest->{self::TENANT_ID_KEY} = $tenantId;
        }
    }
    
    /**
     * 获取租户ID（从 Request 中读取）
     */
    public static function getTenantId(): ?int
    {
        if (self::$currentRequest === null) {
            return null;
        }
        return self::$currentRequest->{self::TENANT_ID_KEY} ?? null;
    }
    
    /**
     * 忽略租户隔离
     */
    public static function ignore(): void
    {
        if (self::$currentRequest !== null) {
            self::$currentRequest->{self::IGNORE_TENANT_KEY} = true;
        }
    }
    
    /**
     * 恢复租户隔离
     */
    public static function restore(): void
    {
        if (self::$currentRequest !== null) {
            self::$currentRequest->{self::IGNORE_TENANT_KEY} = false;
        }
    }
    
    /**
     * 是否应用租户隔离
     */
    public static function shouldApplyTenant(): bool
    {
        if (self::$currentRequest === null) {
            return false;
        }
        
        $ignore = self::$currentRequest->{self::IGNORE_TENANT_KEY} ?? false;
        $tenantId = self::$currentRequest->{self::TENANT_ID_KEY} ?? null;
        
        return !$ignore && $tenantId !== null;
    }
    
    /**
     * 清理上下文（请求结束时调用）
     */
    public static function clear(): void
    {
        self::$currentRequest = null;
    }
}
```

```php
<?php
declare(strict_types=1);

namespace App\Models;

use Framework\Basic\BaseLaORMModel;

class SysUserRole extends BaseLaORMModel
{
    protected $table = 'sys_user_role';
    
    protected $fillable = [
        'user_id',
        'role_id',
        'tenant_id',  // 新增
        'created_by',
        'updated_by',
    ];
    
    protected $casts = [
        'id' => 'integer',
        'user_id' => 'integer',
        'role_id' => 'integer',
        'tenant_id' => 'integer',  // 新增
        'created_by' => 'integer',
        'updated_by' => 'integer',
        'created_at' => 'datetime',
        'updated_at' => 'datetime',
    ];
    
    /**
     * 获取用户在指定租户的角色ID列表
     */
    public static function getRoleIdsByTenant(int $userId, int $tenantId): array
    {
        return self::where('user_id', $userId)
            ->where('tenant_id', $tenantId)
            ->pluck('role_id')
            ->toArray();
    }
    
    /**
     * 同步用户在指定租户的角色
     */
    public static function syncUserRolesByTenant(
        int $userId, 
        int $tenantId, 
        array $roleIds, 
        int $createdBy = 0
    ): void {
        // 只删除该租户下的角色关联
        self::where('user_id', $userId)
            ->where('tenant_id', $tenantId)
            ->delete();
        
        // 插入新的角色关联（带租户ID）
        if (!empty($roleIds)) {
            $data = [];
            $now = now();
            foreach ($roleIds as $roleId) {
                $data[] = [
                    'user_id' => $userId,
                    'role_id' => $roleId,
                    'tenant_id' => $tenantId,
                    'created_by' => $createdBy,
                    'updated_by' => $createdBy,
                    'created_at' => $now,
                    'updated_at' => $now,
                ];
            }
            self::insert($data);
        }
    }
}
```

### 3.3 创建租户表模型

```php
<?php
declare(strict_types=1);

namespace App\Models;

use Framework\Basic\BaseLaORMModel;
use Illuminate\Database\Eloquent\SoftDeletes;

/**
 * 租户模型
 */
class SysTenant extends BaseLaORMModel
{
    use SoftDeletes;
    
    protected $table = 'sys_tenant';
    
    protected $fillable = [
        'tenant_name',      // 租户名称
        'tenant_code',      // 租户编码（唯一）
        'contact_name',     // 联系人
        'contact_phone',    // 联系电话
        'contact_email',    // 联系邮箱
        'status',           // 状态：0=禁用 1=启用
        'expire_time',      // 过期时间
        'max_users',        // 最大用户数
        'remark',           // 备注
        'created_by',
        'updated_by',
    ];
    
    protected $casts = [
        'id' => 'integer',
        'status' => 'integer',
        'max_users' => 'integer',
        'expire_time' => 'datetime',
        'created_by' => 'integer',
        'updated_by' => 'integer',
        'created_at' => 'datetime',
        'updated_at' => 'datetime',
        'deleted_at' => 'datetime',
    ];
    
    // 状态常量
    public const STATUS_DISABLED = 0;
    public const STATUS_ENABLED = 1;
    
    /**
     * 检查租户是否有效
     */
    public function isValid(): bool
    {
        if ($this->status !== self::STATUS_ENABLED) {
            return false;
        }
        
        if ($this->expire_time !== null && $this->expire_time->isPast()) {
            return false;
        }
        
        return true;
    }
    
    /**
     * 检查是否达到最大用户数
     */
    public function isUserLimitReached(): bool
    {
        if ($this->max_users === 0) {
            return false; // 0 表示无限制
        }
        
        $currentUsers = SysUser::whereHas('tenants', function ($query) {
            $query->where('sys_tenant.id', $this->id);
        })->count();
        
        return $currentUsers >= $this->max_users;
    }
}
```

### 3.4 创建用户-租户关联表

```php
<?php
declare(strict_types=1);

namespace App\Models;

use Framework\Basic\BaseLaORMModel;

/**
 * 用户-租户关联模型
 */
class SysUserTenant extends BaseLaORMModel
{
    protected $table = 'sys_user_tenant';
    
    protected $fillable = [
        'user_id',
        'tenant_id',
        'is_default',    // 是否默认租户
        'created_by',
        'updated_by',
    ];
    
    protected $casts = [
        'id' => 'integer',
        'user_id' => 'integer',
        'tenant_id' => 'integer',
        'is_default' => 'boolean',
        'created_by' => 'integer',
        'updated_by' => 'integer',
        'created_at' => 'datetime',
        'updated_at' => 'datetime',
    ];
    
    /**
     * 获取用户的所有租户
     */
    public static function getTenantsByUser(int $userId): array
    {
        return self::where('user_id', $userId)
            ->with('tenant')
            ->get()
            ->toArray();
    }
    
    /**
     * 获取用户的默认租户ID
     */
    public static function getDefaultTenantId(int $userId): ?int
    {
        $record = self::where('user_id', $userId)
            ->where('is_default', true)
            ->first();
        
        return $record ? $record->tenant_id : null;
    }
    
    /**
     * 设置默认租户
     */
    public static function setDefaultTenant(int $userId, int $tenantId): void
    {
        // 取消其他默认
        self::where('user_id', $userId)
            ->where('is_default', true)
            ->update(['is_default' => false]);
        
        // 设置新的默认
        self::where('user_id', $userId)
            ->where('tenant_id', $tenantId)
            ->update(['is_default' => true]);
    }
    
    // 关联
    public function user()
    {
        return $this->belongsTo(SysUser::class, 'user_id');
    }
    
    public function tenant()
    {
        return $this->belongsTo(SysTenant::class, 'tenant_id');
    }
}
```
